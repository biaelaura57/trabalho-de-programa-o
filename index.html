Laços de repetição: são estruturas de programação que executam repetidamente um bloco de código, úteis para automatizar tarefas.

Linguagens de programação: é um conjunto de regras e instruções que permitem criar programas para computadores e outros equipamentos.

Principais tags: usadas para definir parágrafos, títulos, listas, formulários, imagens e links. Head, title, body, p, header, section.

Background_color: é uma propriedade do CSS que define a cor de fundo de um elemento HTML.

Font-Size: Propriedade CSS usada para definir o tamanho da fonte de um texto em um elemento HTML, como font-size: 16px.

Text-Align: Propriedade CSS usada para alinhar o texto dentro de um elemento HTML. Pode ser left, right, center, ou justify.

Text-Transform: Propriedade CSS usada para modificar a aparência do texto, como uppercase, lowercase, capitalize, entre outras.

Console (o que é e para que serve): Ferramenta de desenvolvimento usada para exibir mensagens de erro, logs ou interagir com o código em execução, geralmente encontrada em navegadores ou ambientes de desenvolvimento como o Node.js.
